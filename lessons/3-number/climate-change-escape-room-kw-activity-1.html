🔢 How the Code Works

Each stage reveals one digit of the 5-digit Climate Code.

Digit = |final answer| mod 10

This means: take the absolute value (ignore negatives), then find the last digit.

Example: If your answer is -17, the digit is 7 (because |-17| = 17, and 17 mod 10 = 7)
