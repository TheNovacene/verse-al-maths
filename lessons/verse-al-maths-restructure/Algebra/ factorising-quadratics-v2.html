üß† The Knowing World

We begin with a question: when can a quadratic of the form \(x^2 + bx + c\) be written as \((x + m)(x + n)\)? What must \(m\) and \(n\) do?

Exploration prompts

  • Choose some quadratics: e.g. \(x^2 + 5x + 6\), \(x^2 + 7x + 10\), \(x^2 + 4x + 5\).
  • List all integer pairs \((m, n)\) whose product = \(c\). Which of those also have sum = \(b\)?
  • Notice: if \(m + n = b\) and \(m \cdot n = c\), then \(x^2 + bx + c = (x + m)(x + n)\).
  • Observe: Some quadratics (e.g. \(x^2 + 4x + 5\)) have no integer factor pairs that sum to 4 — those are *not factorable over the integers*.

Formal statement

If \(x^2 + bx + c\) factorises with integer roots, then there exist integers \(m, n\) such that:

  • \(m + n = b\)
  • \(m \cdot n = c\)
  • Hence \(x^2 + bx + c = (x + m)(x + n)\).

Quick check

  • Factorise \(x^2 + 8x + 15\).
  • Does \(x^2 + 6x + 14\) factorise with integer roots? Why / why not?
  • Factorise \(x^2 - 3x + 2\).
üåç The Human Story

Historically, algebraic factorisation has been tied to solving polynomial equations and understanding roots. Mathematicians in the Middle Ages and the Renaissance used methods of factoring to find unknowns and design curves. Even in today’s cryptography and signal processing, factoring polynomials is foundational.
