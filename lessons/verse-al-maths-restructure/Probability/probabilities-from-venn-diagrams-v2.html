Focus Concept: The Addition Rule for Probability

The probability of either event A or event B happening is given by the addition rule:

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

Here, the universal set is a full deck of 52 playing cards.

  • Set A = All Kings
  • Set B = All Black Cards
