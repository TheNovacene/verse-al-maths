🧠 The Knowing World

"To understand the laws that guide, the patterns hidden deep inside."

📘 Key Idea:
The gradient of a line tells us how steep the line is.

Gradient = rise/run = change in y ÷ change in x

🧪 Interactive Sketchpad Activity:

Use this sketch to drag two points on a line and observe how the gradient updates in real time.

🔍 Notice:

  • What happens when one point moves up or down?
  • How does the horizontal distance affect the gradient?

💬 Prompt:
Can you find a line with gradient = 1? What about -2? How do you know?
