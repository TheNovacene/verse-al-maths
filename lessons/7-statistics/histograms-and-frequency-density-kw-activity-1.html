💡 Key Concepts

  • • Frequency Density = Frequency ÷ Class Width
  • • For unequal class widths, bar height shows frequency density, not frequency
  • • The area of each bar is proportional to the frequency
  • • Area = Frequency Density × Class Width = Frequency
